# Color Transform Library and Web Application

This is a companion project for the [Software Resilience and Reliability Book](https://software-resilience-reliability-book.github.io/srr-book/).

## Running the Web Application

From the root directory, run:

```
dotnet run --project src/ColorTransform.Web
```

## Running the Tests

From the root directory, run:

```
dotnet test
```

## Running the Test Harness Sandbox Application

This sandbox project is meant to be run with the debugger to step into the ColorTransform library.
